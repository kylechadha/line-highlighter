Line Highlighter Extension Test Page

Testing Instructions:

  1. Press Alt+L (Option+L on Mac) to enable the highlighter
  2. Click on any text line below to highlight it
  3. Press F to move up one line
  4. Press V to move down one line
  5. Click the extension icon to access settings
  6. Try changing colors and keyboard shortcuts in the popup
